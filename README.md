# B2B Wholesale Platform

A full-stack B2B wholesale e-commerce system built with React, FastAPI, and Tailwind CSS.

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: FastAPI, Python
- Database: PostgreSQL
- Auth: JWT

## Features (Planned)
- Role-based authentication (Admin / Buyer)
- Product catalog with bulk pricing
- Quote-based ordering system
- Admin dashboard for inventory & orders
